Summary

Number of phenotypes scanned: 17 (Phenome-wide association scan is coming soon)
Cohort: UK Biobank
Ancestry: White British
Testing Model for disease: Age as the time scale Cox regression models (y ~ omics genetic score* + strat(sex) + genotyping array + 10PCs)
Testing Model for continuous trait: Ordinary least squares regression (y ~ omics genetic score* + sex + age + clinical center + genotyping array + medication status** + 10PCs)
Selection criterion: FDR-adjusted p-value < 0.05

* Only traits that had a r2>0.01 in internal validation were considerred in the association tests.
** Corresponding medication statuses were used as covariates for traits applicable.
